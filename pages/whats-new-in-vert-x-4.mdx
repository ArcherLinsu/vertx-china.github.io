# What's new in Vert.x 4

Vert.x 4 comes with plenty of new exciting features, some of them have been developed in Vert.x 3 and made available
as tech previews.  Here is an highlight of the most important features supported in Vert.x 4.

## Future model

Vert.x 4 uses futures over callbacks for modelling asynchronous operations.

## Distributed tracing

You can use [Opentracing](https://opentracing.io/) or [Zipking](https://zipkin.io/) to monitor Vert.x microservices and
identify performance issues.

## Reactive SQL clients

A new set of high performance reactive SQL client, fully integrated with Vert.x metrics and Vert.x tracing.
